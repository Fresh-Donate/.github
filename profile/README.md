# FreshDonate — Open Source Donation Platform

FreshDonate — self-hosted donate system for Minecraft servers.

### 🧩 Components:
- [Shop](https://github.com/Fresh-Donate/shop) — Shop page (Nuxt 4).
- [Backend](https://github.com/Fresh-Donate/backend) — System core (Fastify).
- [Panel](https://github.com/Fresh-Donate/panel) — Admin panel (Nuxt 4).
- [Plugin](https://github.com/Fresh-Donate/fresh-donate-plugin) — Plugin for give donates.

### 🛠 Stack:
- **Frontend:** Nuxt 4, Vue 3, Tailwind CSS 4
- **Backend:** Node.js, Fastify, TypeScript
- **Database:** PostgreSQL / Redis
